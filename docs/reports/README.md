# Relatórios do Projeto Skills-Eye

Este diretório contém os relatórios automáticos gerados pelo sistema Alfred.

## Política de Geração

- **Trigger**: Relatórios são gerados apenas após sprint completo e validado
- **Formato**: Markdown (.md)
- **Frequência**: Apenas quando mudanças significativas ocorrerem
- **Conteúdo**:
  - Métricas de qualidade de código
  - Métricas de testes
  - Estilo conciso e objetivo

## Estrutura de Nomenclatura

```
report-{timestamp}-{sprint}.md
```

Exemplo: `report-20251119-sprint-01.md`

## Configuração

As configurações de geração de relatórios estão definidas em:
```
.moai/config/config.json -> reports
```
